## Setup
- Add `@nuxtjs/platform` dependency using yarn or npm to your project
- Add `@nuxtjs/platform` to `modules` section of `nuxt.config.js`

```js
{
  modules: [
    // Simple usage
    '@nuxtjs/platform',

    // With options
    ['@nuxtjs/platform', { /* module options */ }],
 ]
}
```
